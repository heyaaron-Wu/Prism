③ Research Council：从证据到解释

不让研究层直接下结论买卖，而是拆成 Bull / Bear / Risk / Final Judge 四个视角，输出审计解释和人工复核依据。

Research Council Bull / Bear / Risk / Final Judge

🟢 Bull Case

寻找支持继续观察或提高优先级的证据。

  • 趋势与相对强势
  • 行业/事件催化
  • 基本面或估值支撑
  • 外部研究交叉验证

🔴 Bear Case

强制寻找反证，避免单边乐观。

  • 宏观/行业逆风
  • 新闻负面预警
  • 技术破位或量价背离
  • 样本外衰减

🟡 Risk Review

只评估风险，不放宽买入闸门。

  • R:R 是否足够
  • ATR / 波动风险
  • 事件窗口与财报
  • 行业集中度

🔵 Final Judge

输出研究结论和解释，不直接下单。

  • Research verdict
  • Blocker suggestion
  • Shadow difference
  • 人工复核备注

可解释因子层 CAPM / Fama-French / Carhart / R:R

CAPM / Beta

E[Rᵢ] = Rf + βᵢ(E[Rm] - Rf)

用于判断收益来自 alpha 还是市场 beta，适合日报和组合风险归因。

Fama-French 三因子

Rᵢ - Rf = α + βm·MKT + βs·SMB + βv·HML

解释市场、市值、价值暴露；先做归因，不做交易决策。

Carhart 四因子

FF3 + βmom·MOM

增加动量因子，用于判断候选是否过度依赖 momentum。

Risk / Reward

R:R = expected_upside / expected_downside

可作为未来 risk warning / blocker 候选，但目标价来源必须可解释。

⑥ Phase 6 Shadow Lab 与上线闸门

WorldQuant 101、Qlib、非线性预测只作为旁路研究和 shadow score。未来进入生产也只能先成为 blocker、risk flag 或 ranking reference。

Phase 6 Shadow Lab WorldQuant 101 / Qlib / Nonlinear Forecast

WorldQuant 101

rank · delay · ts_rank · correlation · decay_linear

用于旁路 alpha 排序和前向跟踪；不得直接接入 Recommendation active。

Qlib Research

Data → Factor → Model → Backtest → Shadow

Qlib 只作为研究/回测平台，不替代 Prism 生产系统。

Nonlinear Forecast

p_up · p_down · expected_return · confidence · anomaly_risk

只输出概率、置信度和风险，不直接生成 BUY / SELL。
